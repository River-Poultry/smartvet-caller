# Hold Audio

Drop your hold music or advert here as `hold-music.mp3`.

Requirements:
- Format: MP3
- Recommended length: 30–60 seconds (it loops automatically)
- Filename must be exactly: hold-music.mp3

Once the file is in place, it will be served at:
  https://smartvet-caller.onrender.com/audio/hold-music.mp3

The hold TwiML will use it automatically — no code changes needed.
